Uploading Your Roster

In order to determine who is absent from class, the system needs to know who is supposed to be there.

Click on this link to reach the upload roster page. This page can also be reached through the links at the bottom of the page.

  1. Navigate to Aeries
  2. On the left panel of the of the homepage, click on reports and then class roster.

  3. Configure the report to contain the following settings. Make sure to uncheck all boxes on the right. This step is important because the program will not function if the sheet is formatted incorrectly.

  4. Download the report and upload the file to Automatic Attendance Taker. Make sure to bookmark this website for easy access. Now you can start using the app!
