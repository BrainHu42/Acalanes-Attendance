Why Automatic Attendance Taker?

Easy Setup Accurate Efficient
Install and set up the app in less than 5 minutes. Accurately captures live attendance data from Zoom and displays it in an easy-to-read format. Quickly displays attendance data, saving a lot of busy work.

Requirements

  • A roster must be uploaded.
  • Meetings must be started no more than 15 minutes prior to the beginning of class.
  • Read the section on uninvited attendees below

Installation

  1. Click here to download the app
  2. A screen from Zoom will appear prompting you to authorize the app.

    This will allow the app to

    • View meetings to determine the students who are currently present
    • View user profile to verify identity and ensure that only teachers can install

  3. Upon confimation, you will be prompted to register an account

    All that's necessary is to enter a password.

    Your login is automatically your district email.

  4. Now you have to upload your roster.

Uploading Your Roster

In order to determine who is absent from class, the system needs to know who is supposed to be there.

Navigate to this link to reach the upload roster page. This page can also be reached through the links at the bottom of the page.

  1. Go to Aeries
  2. On the left panel of the of the homepage, click on reports and then class roster.

  3. Configure the report to contain the following settings. If it is the first semester, the term code should be 1. If it is the second semester, use term code 3. Make sure to uncheck all boxes on the right. This step is important because the program will not function if the sheet is formatted incorrectly.

  4. After downloading the report and uploading the file to Automatic Attendance Taker, the process is complete. Now you can start using the app!

Uninvited Attendees

Unrecognized names or students with a Zoom username different from their name on Aeries will appear under the uninvited attendees section.

If this person is a student in your class, click confirm and select the correct name of the student if prompted. Click Deny if otherwise.

Note: This step is irreversible and will affect the student's other classes as well.

Once a student has been confirmed during 3 different meetings, the system will remember the student for all subsequent classes.

Note: If a class has already ended, any unconfirmed students will show up as uninvited without the option to confirm.

Uninstallation

  1. Login to your Zoom Account and navigate to the Zoom App Marketplace (or click on this link).
  2. Click Manage and then Installed Apps.
  3. Next to Acalanes Attendance, click the uninstall button.
  4. Upon uninstallation, all data associated with your account will be deleted.
